# Setps to file DS 160 and visa application

1. Fill out DS-160 
    1. If possible find old DS-160 from number e.g (AA96904D) the use it to prefill the new one.
1. Submit it after checking the form for its correctness in its entiriety 
1. Login to https://usvisascheduling.com 
    1. If account does not exist then create a new one. 
1. Go through creating new application and fill out all the info
1. The application will ask for the DS-160 confirmation number. This would be the number filled out in the above step.
1. If filing it with other family members then add it as a dependent.
1. Payment options
    1. can be chosen any option but go back and forth
    1. NEFT and UPI seems to be the easiest one.
